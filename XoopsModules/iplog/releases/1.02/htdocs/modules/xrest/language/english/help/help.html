Help: Xortify

Description

XRest is a API solution for XOOPS and XOOPS Clones, it allows you to have customise plugin functions to call functions on an API to ensure any module can have API Cloud solutions!

Install/uninstall

Please make sure that you download the module from chronolabs to get the latest version, and upload it to your XOOPS Root directory from all the content inside the zips htdocs folder.
