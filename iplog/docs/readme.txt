= Sponsored by Chronolabs Co-op									=
=================================================================

P Log 1.01 is a module for XOOPS 2.5 which allows your XOOPS install to capture and store the IP address and locational information for Users that have logged in for a period of time until they are dropped.

It uses IPInfoDB to get the location of the IP Address as well as storing how many seconds they spent online. It is an admin module only it has no Main User Interface
