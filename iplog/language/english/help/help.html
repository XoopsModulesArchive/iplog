Help: Profile Module

Description

IP Log is a module to record either just Users or both Users and Anonymous guest IP Addresses.

Install/uninstall

No special measures necessary, follow the standard installation process - extract the /htdocs folder into the XOOPS ROOT Path. Install the module through Admin -> System Module -> Modules. If you need detailed instructions on how to install a module, please see the XOOPS Operations Manual.

Make sure you install the IPInfoDB.com API Key in preferences otherwise the module will not function, correctly this is done by going to www.ipinfodb.com and signing up and once you have confirmed your email address you will be provided with your API Key which you paste into the field for it in preferences.
